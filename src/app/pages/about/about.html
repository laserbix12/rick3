¿Qué es esto?

La Rick and Morty API es una API REST (y GraphQL) basada en la serie de televisión Rick and Morty. Tendrás acceso a cientos de personajes, imágenes, ubicaciones y episodios. La Rick and Morty API está llena de información canónica tal como se ve en el programa de televisión.

Consulta la documentación para comenzar
👥

¿Quiénes somos?

Somos Axel Fuhrmann, un chico al que le gusta desarrollar cosas, y Talita, la "científica de datos de Rick and Morty" y fan incondicional de la serie.

🚀

¿Por qué construimos esto?

Porque estábamos muy interesados en la idea de escribir un proyecto de código abierto y además porque Rick and Morty era nuestra serie favorita en ese momento, así que... ¿por qué no?

⚙️

Detalles técnicos

Todo el proyecto está alojado en Digital Ocean y Netlify. El caché de GraphQL se maneja a través de Stellate. Usamos Node y MongoDB para servir la API.

Node.js MongoDB GraphQL Digital Ocean Netlify Stellate
